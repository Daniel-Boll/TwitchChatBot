# Commands 

## Currently working

### Git

The `!git`command will send in chat a specific repo from github based on
the parameters. Parameters are case insensitive.

Possible parameters:
  - (default) : The default parameters expect nothing after `!git`, its full
  command is
  ```
  !git
  ```
  - LambdaIO
  ```
  !git LambdaIO
  ```
  - ChatBot
  ```
  !git ChatBot
  ```

  - Falar
  ```
  !falar <lang> <phrase>
  ```
